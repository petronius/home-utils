Toolbelt

cleanup.py -- Recursively check a directory tree for files that are exact
              duplicates.

hdmesg.pl  -- dmesg(8) output with human-readable time stamps.

md.py      -- Quick markdown converter for nice HTML documents with good code
              highlighting.

slow.sh    -- Simulate a bad network connection on the local machine. You can
              modify bandwidth and respone times. Comes with three presets.

ts.py      -- Read data from stdin, and emit to stdout with a timestamp on each
              line. (The timestamp is the time that the line was read.)

              Most useful with ping(8): `ping <server> | ts`

volume.sh  -- Get ALSA volume status and output to stdout. (Used by Conky)
